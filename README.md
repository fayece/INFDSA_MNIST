# INFDSA_MNIST

An ongoing project exploring machine learning techniques using the MNIST dataset. Each algorithm is implemented from scratch to build understanding. No ML libraries for the creating the models.

## Goal
To understand machine learning from the ground up. Starting at understanding the dataset, then implementing algorithms manually and analyzing the results.

## Constraints
Models are built under strict constraints to force genuine understanding of the algorithms:
- **No external libraries**: Models are implemented from scratch using only NumPy. ML libraries may be used during dataset exploration, but are strictly prohibited starting at the creation of models.
- **Target: embedded system**
  - **Memory**: 256 KB
  - **Storage**: 1 MB
  - **GPU**: None

## Approach
1. Dataset analysis: profiling and exploring the data before taking action
2. Decision tree: baseline model
3. Further algorithms to come
